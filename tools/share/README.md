# 分享卡片生成器

一个现代化的单页应用，用于将链接转换成精美的分享卡片图片。

## 功能特点

### 卡片类型

1. **音乐卡片** 🎵
   - 播放器风格设计
   - 显示歌名、歌手、封面
   - 可选喜爱歌词片段
   - 支持自动获取链接元数据

2. **播客卡片** 🎙️
   - 播客播放器样式
   - 显示节目名、播客名、封面
   - 可选节目简介
   - 支持自动获取链接元数据

3. **文章卡片** 📄
   - 展示文章标题和作者/来源
   - 可选150字摘要（支持OpenAI自动生成）
   - 可选金句摘抄
   - 适合分享阅读内容

4. **纯文字卡片** 📝
   - 纯文字内容展示
   - 支持文字高亮功能
   - 适合分享想法和语录

### 主题样式

提供6种精美主题：

- **现代** - 渐变紫色，时尚现代
- **经典** - 灰白配色，经典大方
- **极简** - 纯白简洁，清爽利落
- **鲜艳** - 粉色渐变，活力四射
- **优雅** - 深色高级，优雅大气
- **复古** - 米黄配色，复古怀旧

### 核心功能

- ✨ 拟物化设计，简约又不失细节
- 📱 iPhone友好的响应式UI（特别优化iPhone 15比例）
- 🎨 6套精美卡片主题可选
- 🔗 自动获取链接元数据（音乐/播客）
- 🤖 OpenAI API集成，智能生成文章摘要
- 💾 一键下载高清卡片图片
- ⚙️ 可配置的设置面板

## 使用说明

### 1. 选择卡片类型

在首页选择你想创建的卡片类型（音乐、播客、文章或纯文字）。

### 2. 填写内容

根据卡片类型填写相应信息：
- 必填项用 * 标注
- 支持输入链接自动获取元数据（音乐/播客）
- 文章卡片可选择自动生成摘要

### 3. 预览并选择样式

点击"预览卡片"后：
- 查看卡片效果
- 切换不同主题样式
- 实时预览变化

### 4. 下载卡片

满意后点击"下载卡片"按钮，系统将生成高清PNG图片并自动下载。

## 设置配置

点击右上角⚙️图标进入设置面板：

- **OpenAI API Key** - 用于自动生成文章摘要（可选）
- **API 代理地址** - 如需使用代理访问OpenAI API（可选）

设置保存在浏览器本地，下次访问时自动加载。

## 技术栈

- 纯前端实现，无需后端服务
- 模块化JavaScript架构
- CSS3 现代样式设计
- html2canvas 用于卡片图片生成
- localStorage 用于设置持久化

## 浏览器兼容性

推荐使用现代浏览器：
- Chrome/Edge 90+
- Safari 14+
- Firefox 88+

## 隐私说明

- 所有数据处理在本地浏览器完成
- 不上传任何用户数据到服务器
- OpenAI API Key 仅存储在本地浏览器
- 链接元数据获取通过CORS代理服务

## 开发

本应用为纯静态页面，可直接在任何Web服务器上运行。

